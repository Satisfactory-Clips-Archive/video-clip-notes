> Will there be MODS? So we had actually been planning MOD support from the very beginning and we even have the foundations laid down for it but what we found was it was adding so much development time to Satisfactory and what we really thought we needed right now is to get the base game right get it tight, robust and running smoothly So because it was adding so much development time. We kind of stopped working on it so we don't have support for right now.  We can't really promise that we will have it now or in the future But it's obviously a really good idea. So we're gonna see what we can do. General plans for Post-Launch? So people have asked questions about like DLCs, Loot Boxes, Micro Transactions ....
